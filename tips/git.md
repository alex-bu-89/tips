# Git
### Commit message conventions
```
- [feat] (feature)
- [fix] (bug fix)
- [docs] (documentation)
- [style] (formatting, missing semi colons, â€¦)
- [refactor]
- [test] (when adding missing tests)
- [chore] (maintain)
```
### Pretty git log
```
git log --format="%C(auto) %h %d %aI %an %s"
```
